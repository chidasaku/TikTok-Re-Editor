# TikTok Re-Editor

テキストから透過動画を自動生成するアプリ

---

## できること

### 入力
- **動画から生成** - 動画をアップロードして自動文字起こし
- **ファイルから生成** - テキストファイル(.txt)を読み込み
- **テキスト入力** - 直接テキストを入力

### テキスト処理
- **自動整形** - 14文字/行に自動整形
- **ひらがな変換** - 音声生成用にひらがなに変換（読み間違い防止）
- **ファイル名自動生成** - テキスト内容からファイル名を生成
- **編集機能** - 整形後のテキストを手動で編集可能

### 音声生成（VOICEVOX）
- **キャラクター選択** - 青山龍星、ずんだもん等
- **スタイル選択** - ノーマル、喜び、悲しみ等
- **話速調整** - 0.7x〜1.3x
- **間の長さ調整** - 句読点での間隔を調整
- **プレビュー** - 生成前に声を確認

### 動画生成
- **透過動画出力** - MOV（ProRes 4444）
- **解像度** - 1080 x 1920（縦動画）
- **プレビュー** - iPhone風フレームで確認

### SNS用コンテンツ生成
- **タイトル案** - 3パターン生成
- **紹介文案** - 3パターン生成
- **ハッシュタグ** - 5つ生成

### ダウンロード
- テキスト（TXT）、音声（WAV）、動画（MOV）

## セットアップ

### 1. VOICEVOXをインストール

https://voicevox.hiroshiba.jp/

### 2. インストールスクリプトを実行

| OS | ダブルクリック |
|----|---------------|
| Mac | `インストール.command` |
| Windows | `インストール.bat` |

Python、FFmpeg、必要なパッケージが自動でインストールされます。

### 3. アプリを起動

1. VOICEVOXを起動
2. 起動ファイルをダブルクリック

| OS | ダブルクリック |
|----|---------------|
| Mac | `起動.command` |
| Windows | `起動.bat` |

ブラウザが自動で開きます（http://localhost:8501）

## 使い方

### 流れ

```
テキスト入力 → 整形 → ひらがな変換 → 音声生成 → 動画生成
```

### 1. 入力方法を選択

3つの方法から選べます：

| タブ | 説明 | 必要なAPI |
|------|------|-----------|
| 動画から生成 | 動画をアップロードして文字起こし | Gladia + Gemini |
| ファイルから生成 | テキストファイル(.txt)を読み込み | Gemini |
| テキスト入力 | 直接テキストを入力 | Gemini |

- 入力後「START」をクリック
- AIが14文字/行に自動整形します

### 2. テキストを編集

画面に2つのテキストエリアが表示されます：

| 左側 | 右側 |
|------|------|
| 整形済みテキスト | ひらがなテキスト |
| 動画に表示される文字 | 音声の読み方 |

- 「ひらがなに変換」ボタンで右側にひらがなが生成されます
- どちらも編集可能です

### 3. 音声を生成

- キャラクターを選択（青山龍星など）
- 話速を調整（0.7x〜1.3x）
- 間の長さを調整（句読点での間隔）
- 「GENERATE AUDIO」をクリック

※ひらがなテキストがあれば自動的にそちらを使用します

### 4. 動画を生成

- 「GENERATE VIDEO」をクリック
- 完了まで待つ（テキスト量により数分かかる場合があります）

### 5. ダウンロード

- プレビューで確認
- 「DOWNLOAD VIDEO」をクリック
- MOVファイルが保存されます

### ひらがな変換について

VOICEVOXは漢字や数字の読み方を間違えることがあります。

例：
- 「2026年」→「にまるにろくねん」（間違い）
- 「にせんにじゅうろくねん」（正しい）

ひらがなに変換することで、正確な読み上げが可能になります。

## API設定

| API | 用途 | 取得先 |
|-----|------|--------|
| Gemini | テキスト整形・ひらがな変換 | https://aistudio.google.com/apikey |
| Gladia | 動画から文字起こし（任意） | https://www.gladia.io/ |

アプリ上部の「⚙️ API設定」から入力・保存できます。

## 出力形式

- MOV（ProRes 4444）
- 1080 x 1920（縦動画）
- 背景透過

## トラブルシューティング

| エラー | 解決方法 |
|--------|---------|
| VOICEVOXに接続できません | VOICEVOXを起動してください |
| ひらがな変換に失敗 | Gemini APIキーを確認 |
| ffmpeg not found | インストールスクリプトを再実行 |

## 手動インストール

インストールスクリプトがうまくいかない場合：

**Mac:**
```bash
brew install python ffmpeg
pip install -r requirements.txt
```

**Windows:**
1. Python: https://www.python.org/downloads/ （Add to PATHにチェック）
2. FFmpeg: https://github.com/BtbN/FFmpeg-Builds/releases
3. `pip install -r requirements.txt`
