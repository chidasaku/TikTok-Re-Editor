# TikTok Re-Editor (VOICEVOX版)

動画の文字起こし → テキスト整形 → 音声合成 → **透過動画生成**を自動実行するアプリ

---

## ステップ1: 必要なソフトをダウンロード

以下の3つをダウンロードしてインストールしてください。

### 1. Python（パイソン）
プログラムを動かすために必要です。

1. https://www.python.org/downloads/ にアクセス
2. 「Download Python 3.xx」ボタンをクリック
3. ダウンロードしたファイルを開いてインストール

【重要】インストール時に「Add Python to PATH」に必ずチェックを入れてください

---

### 2. VOICEVOX（ボイスボックス）
音声を作成するために必要です。

1. https://voicevox.hiroshiba.jp/ にアクセス
2. 「ダウンロード」ボタンをクリック
3. お使いのOS（Windows / Mac）を選択
4. ダウンロードしたファイルを開いてインストール

---

### 3. FFmpeg（エフエフエムペグ）
動画を作成するために必要です。

#### Macの場合

1. 「ターミナル」アプリを開く（Finder → アプリケーション → ユーティリティ → ターミナル）

2. 以下のコマンドをコピーして貼り付け、Enterキーを押す：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

【パスワード入力について】
- 「Password:」と表示されたら、Macのログインパスワードを入力
- 入力しても画面に文字は表示されません（セキュリティのため）
- 見えなくても入力されているので、打ち終わったらEnterを押してください

3. 完了したら、以下のコマンドをコピーして貼り付け、Enterキーを押す：

brew install ffmpeg

#### Windowsの場合

1. https://github.com/BtbN/FFmpeg-Builds/releases にアクセス
2. 「ffmpeg-master-latest-win64-gpl.zip」をダウンロード
3. ZIPファイルを解凍
4. 解凍したフォルダを「C:\ffmpeg」に移動
5. 環境変数にパスを追加：
   - 「スタート」→「環境変数」で検索
   - 「システム環境変数の編集」を開く
   - 「環境変数」→「Path」→「編集」→「新規」
   - 「C:\ffmpeg\bin」を追加して「OK」

---

## ステップ2: アプリをセットアップ

### 1. ZIPファイルを解凍

ダウンロードした「TikTok-Re-Editor-FFmpeg.zip」をダブルクリックして解凍します。

### 2. 必要なパッケージをインストール

#### Macの場合:

1. 解凍した「TikTok-Re-Editor-FFmpeg」フォルダを**右クリック**
2. メニューの一番下にある「**新規ターミナルでフォルダに移動**」をクリック
3. ターミナルが開いたら、以下のコマンドを入力してEnter：

pip install -r requirements.txt

#### Windowsの場合:

1. 解凍した「TikTok-Re-Editor-FFmpeg」フォルダを開く
2. アドレスバー（上部のパス表示部分）をクリックして「cmd」と入力してEnter
3. コマンドプロンプトが開いたら、以下のコマンドを入力してEnter：

pip install -r requirements.txt

### 3. 設定ファイルを作成

1. フォルダ内の「.env.example」ファイルを探す

【ファイルが見つからない場合】
Macでは「.」で始まるファイルは隠しファイルです。
キーボードで「Cmd + Shift + .」（ピリオド）を同時に押すと表示されます。

2. 「.env.example」をコピーして、名前を「.env」に変更

【APIキーについて】
- テキスト入力だけで使う場合、APIキーは不要です
- .envファイルはそのままでOK（プレースホルダーは自動で無視されます）
- APIキーはアプリ画面上部の「API設定」からも入力できます

---

## ステップ3: アプリを起動

### 1. VOICEVOXを起動

まず、VOICEVOXアプリを起動してください。起動したままにしておきます。
（起動に少し時間がかかります）

### 2. アプリを起動

フォルダ内の起動ファイルをダブルクリック！

| OS | ファイル名 |
|----|-----------|
| Mac | 起動.command |
| Windows | 起動.bat |

3秒後に自動的にブラウザが開きます。

【Macで「開発元を確認できない」と出た場合】
1. 「システム設定」→「プライバシーとセキュリティ」を開く
2. 下にスクロールして「このまま開く」をクリック

【ブラウザが自動で開かない場合】
手動でブラウザを開いて以下にアクセス：
http://localhost:8501

## 使い方

### 基本的な流れ

テキスト入力 → 音声生成 → 動画生成 → ダウンロード

### 手順

1. テキストを入力
   - 「テキスト入力」タブを選択
   - テキストを貼り付けて「START」

2. 音声を生成
   - キャラクターを選択
   - 話速・間の長さを調整
   - 「GENERATE AUDIO」をクリック

3. 動画を生成
   - 「GENERATE VIDEO」をクリック
   - 完了までしばらく待つ

4. ダウンロード
   - 「DOWNLOAD VIDEO」をクリック
   - MOVファイルが保存されます

## 🆕 新機能：読み方と表示を別々に設定

音声の読み方と、動画に表示するテキストを別々に設定できます。

### こんな時に便利

- 難しい漢字を正しく読み上げたい
- 人名・地名の読み方を指定したい
- ひらがなで発音を調整したいが、表示は漢字にしたい

### 使い方

#### ステップ1：ひらがなで音声を作る

```
例：「きょうはいいてんきです」
```

1. テキストを**ひらがな**で入力（正しい読み方で）
2. 「GENERATE AUDIO」をクリック
3. 音声を確認 ✅

**この時点で音声用のテキストが保存されます**

#### ステップ2：漢字に書き換える

```
例：「今日は良い天気です」
```

1. テキストエリアの内容を**漢字表記**に変更
2. これが動画に表示されるテキストになります

#### ステップ3：動画を生成

1. 「GENERATE VIDEO」をクリック
2. 結果：
   - 🔊 音声 → ひらがなの読み方で再生
   - 📝 表示 → 漢字で表示

### 具体例

| 手順 | テキスト内容 | 何が起こる |
|------|-------------|-----------|
| ① 音声生成 | 「さとうたろう」 | この読み方で音声が作られる |
| ② テキスト変更 | 「佐藤太郎」 | 表示用に変更 |
| ③ 動画生成 | - | 音声は「さとうたろう」、表示は「佐藤太郎」 |

### 注意点

- **行数は同じにしてください**（音声用と表示用で行数が違うと警告が出ます）
- 音声を再生成すると、その時点のテキストで上書きされます
- テキストを変更すると「⚠️ 変更あり: ○行目」と表示されます

## よくあるエラーと解決方法

### 「VOICEVOXに接続できません」

原因: VOICEVOXアプリが起動していない

解決方法:
1. VOICEVOXアプリを起動する
2. 起動完了まで待つ（数秒かかります）
3. ブラウザをリロード

---

### 「pip: command not found」

原因: Pythonが正しくインストールされていない

解決方法:
1. Pythonを再インストール
2. インストール時に「Add Python to PATH」にチェックを入れる
3. パソコンを再起動

---

### 「streamlit: command not found」

原因: streamlitがインストールされていない

解決方法:
ターミナルで以下を実行：
pip install streamlit

---

### 「ffmpeg: command not found」

原因: FFmpegがインストールされていない

解決方法:
- Mac: ターミナルで「brew install ffmpeg」を実行
- Windows: 環境変数の設定を確認

---

### 「PermissionError」または画面が真っ白になる（Mac）

原因: macOSのセキュリティ制限（ダウンロードしたファイルの隔離フラグ）

解決方法:
1. 「ターミナル」アプリを開く
2. 以下のコマンドを実行（フォルダのパスは実際の場所に置き換えてください）：

xattr -cr ~/Desktop/TikTok-Re-Editor-FFmpeg

3. もう一度「起動.command」をダブルクリック

---

### 動画生成が途中で止まる

原因: テキストが長すぎる、またはメモリ不足

解決方法:
1. テキストを短くして試す
2. 他のアプリを閉じてメモリを確保
3. パソコンを再起動

## 機能一覧

| 機能 | 説明 |
|------|------|
| 動画から文字起こし | 動画をアップロードして自動で文字起こし（APIキー必要） |
| テキスト整形 | 改行ごとに句読点を自動追加 |
| 音声合成 | VOICEVOXで自然な音声を生成 |
| 話速調整 | 0.7x〜1.3xで速度を変更 |
| 間の長さ調整 | 句読点での間隔を調整（0.00〜2.00） |
| 透過動画出力 | Premiere Proで使える透過MOV |

---

## 出力形式

- 形式: MOV（ProRes 4444）
- 解像度: 1080 x 1920（縦動画）
- 特徴: アルファチャンネル対応（背景透過）
- 用途: Premiere Pro、Final Cut Pro等で透過素材として使用可能

---

## API設定（オプション）

動画から文字起こしする場合のみ必要です。
テキスト入力だけで使う場合は不要です。

APIキーはアプリ画面上部の「⚙️ API設定」をクリックして入力できます。
（.envファイルに書いておくこともできます）

| API | 用途 | 取得先 |
|-----|------|--------|
| Gladia | 文字起こし | https://www.gladia.io/ |
| Gemini | テキスト整形 | https://aistudio.google.com/apikey |

---

## ヒント

- テキスト入力だけで使う場合、APIキーは不要です
- VOICEVOXは起動に少し時間がかかります
- 長いテキストは複数のクリップに分割されます
- プレビューで確認してからダウンロードできます
- アプリを終了するには、ターミナルウィンドウを閉じてください
